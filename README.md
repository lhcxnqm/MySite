# MySite
基于Django的个人博客

#### 已添加功能
* 用户登录、注册
* 首页图片轮播
* 博客分类、内容分页
* 文章评论，富文本编辑
* 访问统计、热门文章推荐
* 豆瓣新片推荐（静态爬取）
* 1905网"今日影评"获取（Json解析）
* 12306登录（Selenium模拟访问）

#### Tips
* 合理套用母版页，减少代码量
* 界面通过BootStrap美化
* 运用HighCharts工具进行数据图表化
* 小图标使用font-awesome样式
